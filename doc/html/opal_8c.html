Print formatted message to log file.

Helper function to log messages. Function writes to global variable log_fd. Usually called by a macro logger. Eg:

logger (ERROR, "Cannot read file: %s", file_name);

logger (DEBUG, "access('%s', F_OK)", source_fn););

Parameters
[in]tagLog level of message
[in]fileSource file name
[in]lineSource file line number
[in]lineSource file function
[in]fmtFormatted message to log
Returns
None
  1. Assert log file descriptor is not null
  2. Allocate buffer to hold message to log
  3. Read formatted user message string into the buffer

  4. If tag is a result of a system call and current log level is more than DEBUG, print the message and return.

    Eg - PASS / FAIL etc

  5. If log message is less than current log level, print message with current date, time to log file descriptor. Eg.

    [05/02/2021 20:57:58] [DEBUG] main() [source_fd] access('input/hello.opl', R_OK) - PASS

  6. Flush message to log file
